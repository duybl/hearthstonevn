Headers With Style

No Header but Article Image

Feeling Responsive enables you to get the attention of visitors. If you don't want to use a big header, use an image for the article instead.

First turn of the header with header: no. Then add an image to with the following code. The caption is optional. 

header: no
image:
    title: unsplash_eagle.jpg
    caption: This is a caption for the header image with link
    caption_url: https://unsplash.com/
